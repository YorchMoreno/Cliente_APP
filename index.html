Instalar ClienteAPP
En tu escritorio o dispositivo
¡Tu navegador soporta instalación directa!
Haz clic en el botón para instalar ahora

Sigue estos pasos según tu navegador:

🌐 Chrome o Edge (PC / Android)
  1. Abre la app en Chrome o Edge
  2. Haz clic en los 3 puntos (menú) arriba a la derecha
  3. Selecciona "Instalar ClienteAPP" o "Guardar e instalar"
  4. Confirma la instalación
🍎 Safari (iPhone / iPad)
  1. Abre la app en Safari
  2. Toca el botón Compartir
  3. Selecciona "Agregar a pantalla de inicio"
  4. Toca Agregar
Importante: La app debe estar en un servidor web (no abrirse como archivo local) para que la instalación funcione correctamente. Si la abres con file://, usa un servidor local como Live Server en VS Code.
